INSTRUCTIONS

  1. Install the Arduino IDE
  2. Plug the Arduino
  3. In the IDE go to
    1. Files
    2. Examples
    3. Firmata
    4. Standard Firmata
  4. Go to
    1. Tools
    2. Board: Choose the one you have (it should be there automatically)
    3. Port: choose it (should select it automatically as well)
  5. Press Verify (up and to the left)
  6. Then Upload (right next to Verify)
  7. And your Arduino is ready to rock with Johnny Five!
